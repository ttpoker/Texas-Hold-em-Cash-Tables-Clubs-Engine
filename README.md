## 一、项目亮点

- **现金桌系统**  
  - 支持 6~9 人现金桌，自动配桌、自动结算  
  - 支持多桌并行、多盲注级别、多底池结构  
- **俱乐部与联盟系统**  
  - 俱乐部创建、成员管理、俱乐部专属房间  
  - 代理解/分润模型，支持多层级代理解结构  
  - 俱乐部专属赛事、私局/朋友局支持  
- **安全与风控**  
  - 内置防作弊提示结构（可扩展反串通、反机器人、反多开逻辑）  
  - 适合本地/区域平台，可与风控/反作弊系统对接  
- **可扩展性**  
  - 模块化设计，支持后续扩展：  
    - SNG / MTT 赛事模式  
    - 短牌（Short Deck）玩法  
    - 更复杂的返水/返佣/积分系统
